🔗 Share this tracker

The easiest way to share is via a free hosting service. Your data stays in the file — just re-upload whenever you add new entries.

1
Export your data — click Export above to download linguatrack-data.json. This keeps all your additions safe.
2
Host for free on Netlify Drop — go to app.netlify.com/drop, then drag and drop this HTML file. You'll get a permanent URL instantly (no account needed).
3
Or use GitHub Pages — upload the HTML file to a GitHub repo, enable Pages in Settings → Pages. Free permanent URL at username.github.io/repo-name.
4
Share the URL with your wife. She can use the Import button to load the JSON data file, and Export to save any additions she makes.
Pro tip
Both of you can keep a local copy of the JSON file and use Import/Export to sync. Think of it like a shared Google Doc but fully offline and private.
